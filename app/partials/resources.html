Resources

Active Local Coverage Determinations (LCDs)
Use this to check the allowable amounts for various medical supplies and product therapies. You can also enter the HCPCS code you are looking for in the search bar on the top of the page.
HCPCS code and modifier descriptions
Simply enter the HCPCS code or modifier you are looking into for a brief description. You can also look up the fee schedule for any HCPCS code.
Common medical abbreviations and ICD codes
Search for common medical abbreviations that are used by physicians. This is useful when reviewing chart notes. You can also search for ICD codes if it is unclear what the diagnosis should be for a Medicare beneficiary.
NPI Registry search
Use this to verify the NPI number for a physician. It includes demographic information along with a phone and fax number.
Competitive Bidding Area (CBA) search
Enter the beneficiary's zip code to verify if they are in a competitive bidding area. You can then determine if they are in a CBA you are not contracted with.
